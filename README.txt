1. 일주일 무료 사용을 통해, 사용자에게 MAC 주소를 받습니다.
2. 주소를 기반으로 사용자를 제한합니다.
3. 사용 시간 제한을 설정합니다. (30일 기준)
4. 배포하기 전에 'iherb URL header'를 무작위로 변경합니다.
5. 제품 사진 추출은 되도록이면 10분에 한 번만 실행하는 것을 추천해줍니다.

# MAC 주소 확인하는 법

[ Windows ]

1. '시작'에서 'cmd'를 검색한 후 실행합니다.
2. cmd 창에서 'ipconfig /all'을 입력하고 Enter를 누릅니다.
3. Network Adapter의 '물리적 주소(MAC 주소)'를 확인합니다.

[ MacOS ]

1. 'Launchpad'에서 'Terminal'을 찾아 실행합니다.
2. Terminal 창에서 'ifconfig | grep ether'를 입력하고 Enter를 누릅니다.
3. 'ether' 다음에 표시되는 값이 'MAC 주소'입니다.
